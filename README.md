# 📊 Análise de Infecções nos Municípios de SP (2013 - 2023)

Este projeto foi desenvolvido para analisar a evolução histórica das quantidades de infecções registradas nos municípios e no estado de São Paulo. 

## 🛠️ Tecnologias Utilizadas
- **Python** (Linguagem principal)
- **Pandas** (Tratamento e manipulação de dados)
- **Google Colab** (Ambiente de desenvolvimento)

## 📈 O que foi analisado?
- Filtro e agrupamento dos anos de 2013 e 2023.
- Identificação dos municípios com maior volume de casos.
- Criação de gráficos para melhor visualização e entendimento.
- Análises dos municípios com o maior numero de infecção por 100 mil habitantes.
